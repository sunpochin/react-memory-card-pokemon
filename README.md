# A React.js memory card game

https://sunpochin.github.io/react-memory-card-pokemon/

## Installation
```
git clone git@github.com:sunpochin/react-memory-card-pokemon.git
cd react-memory-card-pokemon

npm install
npm start
```

## Update github page App
npm run deploy
